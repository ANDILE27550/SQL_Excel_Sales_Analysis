Project Description
This project demonstrates how SQL queries can be used to extract sales data and prepare it for analysis in Excel. The project focuses on transforming raw data into structured tables and simple reports.

## Tools Used
- MySQL 8.0 Command Line Client
- Microsoft Excel

## SQL Skills Demonstrated
- SELECT statements
- Aggregate functions
- GROUP BY
- ORDER BY

## Excel Skills Demonstrated
- Data cleaning
- Table creation
- Basic calculations
- Data formatting

## Key Insights
- Sales totals were calculated.
- Product performance was analyzed.
- Clean datasets were prepared for reporting.

## Outcome
This project shows the ability to combine SQL querying with Excel analysis, a common requirement in real-world data analysis and freelancing projects.
